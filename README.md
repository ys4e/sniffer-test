# sniffer-test

A demo application for [ys-sniffer](https://github.com/ys4e/ys-sniffer).

## Usage

```sh
cargo run --release
```

Any packets received will be dumped to the `dump/` folder.\
Seeds which are brute-forced will be found in `known_seeds.txt`.
